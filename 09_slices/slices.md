#### In Rust, `Vec<T>` is the dynamic growable array equivalent of Go slices. A helper function formats output as `[a b c]` matching Go's space-separated style.
___
##### Run Command:

`$ cargo run`

##### Results:

`emp: [  ]`
`set: [a b c]`
`get: c`
`len: 3`
`apd: [a b c d e f]`
`cpy: [a b c d e f]`
`sl1: [c d e]`
`sl2: [a b c d e]`
`sl3: [c d e f]`
`dcl: [g h i]`
`2d:  [[0] [1 2] [2 3 4]]`
___
#### Check out the Rust documentation on [slices](https://doc.rust-lang.org/std/primitive.slice.html) and [Vec](https://doc.rust-lang.org/std/vec/struct.Vec.html) for more details.
